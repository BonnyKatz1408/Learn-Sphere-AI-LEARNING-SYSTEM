LearnSphere
AI-Powered Personalized Learning System

---Team GreenHorizon

 
Executive Summary
The Adaptive Learning Platform is an intelligent educational system that revolutionizes personalized learning through AI-driven content generation, adaptive assessments, and multi-modal content delivery. The platform features two core modules—Learn and Quiz—designed to provide students, professionals, and educators with customized learning experiences that adapt to individual needs and learning styles.
Key Innovation Highlights
Feature	Description
Persona-Based Learning	Three adaptive personas (Student, Professor, Professional) with tailored complexity levels
Adaptive Roadmapping	AI-generated learning paths with phase-based progression
Infinite Assessment	Unlimited question generation with intelligent feedback
Multi-Modal Delivery	TTS audio, downloadable lectures, and code generation
Real-Time Analytics	Mastery tracking, confidence metrics, and time optimization

Platform Architecture
System Overview
The platform is built on a modern web architecture with secure authentication, database-backed user management, and AI-powered content generation. The system integrates multiple technologies to deliver a seamless, responsive learning experience.
Core Modules
1. LEARN Module
The Learn module serves as the primary educational engine, providing comprehensive, adaptive learning experiences tailored to individual user profiles.

Adaptive Roadmap Generation
Upon topic submission, the system generates a comprehensive learning roadmap structured into multiple phases, each covering distinct subtopics with progressive difficulty. The roadmap intelligently sequences content to build foundational knowledge before advancing to complex concepts.
Interactive Skill Dependency Map
The dependency map provides a visual representation of skill relationships and prerequisites. Users can:
•	Mark individual skills as complete or incomplete to track progress
•	Click on any skill node to access curated resources including videos and articles
•	Visualize prerequisite relationships to understand optimal learning sequences
•	Access integrated search functionality for supplementary materials
Learning Metrics Dashboard
Metric	Description	Purpose
Mastery Completion	Percentage of topics mastered	Track overall progress toward topic expertise
Time Saved	Efficiency gained through adaptive learning	Measure learning optimization vs. traditional methods
Confidence Score	Self-assessed understanding level	Identify areas requiring additional focus
Coverage	Breadth of subtopics completed	Ensure comprehensive topic coverage
 
Infinite Assessment Engine
The assessment engine leverages AI to generate unlimited, contextually relevant questions for continuous practice and evaluation. The system provides:
•	Instant Feedback: Correct answers highlighted in green; incorrect answers trigger detailed explanations
•	Weakness Identification: AI analyzes patterns to pinpoint knowledge gaps and recommend focus areas
•	Adaptive Difficulty: Questions dynamically adjust based on user performance
•	Comprehensive Coverage: Ensures questions span all subtopics and difficulty levels
Intuition: Deep Conceptual Understanding
The Intuition tab delivers persona-adapted textual explanations that build conceptual understanding beyond rote memorization. Complexity and explanation depth automatically adjust to match the selected persona, ensuring optimal comprehension.
Multi-Accent Text-to-Speech System
Powered by Microsoft Edge TTS, the platform converts all generated text into natural-sounding audio with four accent options. Features include:
•	Real-time playback with pause/resume controls
•	Downloadable audio files for offline learning
•	Multi-accent support for diverse learning preferences
•	Synchronized text highlighting during playback
Code Lab: Interactive Programming Environment
The Code Lab generates detailed Python implementations with extensive inline comments explaining every component. Each code block is optimized for learning with:
•	Hundreds of explanatory comments annotating logic, algorithms, and best practices
•	One-click copy and download functionality
•	Integrated code search engine for alternative implementations
•	Syntax highlighting and formatting for readability
Audio Guide: Downloadable Lectures
Transform any generated learning content into professional audio lectures. Users can download complete topic explanations as structured audio files, enabling learning during commutes, workouts, or other activities.
Follow-Up Question Buddy
An integrated AI assistant addresses clarification questions in real-time, providing instant responses to user queries about the learning material. This creates an interactive learning experience similar to having a personal tutor.
 
2. QUIZ Module
The Quiz module provides standalone assessment capabilities with the same infinite question generation and intelligent feedback systems as the Learn module. It serves as a rapid evaluation tool for knowledge retention and concept mastery.

Frontend Technologies

•	Interactive visualizations for skill dependency mapping
•	Real-time progress updates and metric dashboards
